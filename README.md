# Memory Cards PRO 🧠

Memory Cards PRO is a modern flashcard application built with **HTML**, **CSS**, and **Vanilla JavaScript**.

Users can create their own question-and-answer cards, flip them with a smooth animation, search through saved cards, and keep everything stored in the browser using localStorage.

## ✨ Features

- Create custom memory cards
- Flip cards to reveal answers
- Delete cards
- Search cards by question or answer
- Clear search input
- Total cards counter
- Empty state
- Smooth 3D flip animation
- localStorage support
- Responsive layout

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript ES6+
- localStorage
